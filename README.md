# Muskan Wires Website

Official website repository for Muskan Wires.

## Overview

Muskan Wires is a manufacturer and supplier of wire products serving industrial, agricultural, and infrastructure sectors.

This repository contains the source code and deployment configuration for the Muskan Wires website.

## Technology Stack

* Frontend: Bolt.new
* Hosting: Netlify
* Database: Supabase
* CRM: Zoho CRM (Planned)
* Automation: n8n (Planned)
* AI Assistant: OpenAI / Gemini (Planned)

## Features

* Responsive website
* Product catalog
* Lead capture forms
* Contact inquiry system
* Supabase integration
* SEO optimized pages
